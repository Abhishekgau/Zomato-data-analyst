# Zomato Data Analytics Project 🍔

A Python-based data analysis project focused on exploring restaurant performance, customer preferences, and pricing trends using the Zomato dataset.

---

## 🛠️ Python Libraries Used
* **Pandas:** For data ingestion, cleaning, and manipulation.
* **NumPy:** For efficient numerical operations.
* **Matplotlib & Seaborn:** For exploratory data analysis (EDA) and plotting trend charts.

## 💡 Key Analysis & Insights
* **Data Cleaning:** Handled null values, treated duplicates, and converted pricing data into clean numerical formats.
* **Cuisine Demand:** Analyzed which cuisines are highest rated and most heavily ordered across different locations.
* **Cost vs Rating:** Explored how the average cost for two people impacts overall consumer ratings.
* **Online Booking Impact:** Evaluated how offering online table bookings changes restaurant visibility and success.
